# 新增：对比模式（不同 System Prompt 的 Tool Call 策略对比）

## 目标

在自由模式中新增"对比模式"tab，同一个问题用 3 种不同的 System Prompt 同时运行，
并列展示 Tool Call 策略的差异。

## 改动范围

### 新增文件

`demo/src/engine/comparisonAgent.ts`
- 类似 LiveAgent，但同时运行 3 个实例（共用 API Key/Model/BaseUrl）
- 每个实例使用不同的 systemPrompt
- 3 个实例并行执行，各自维护独立的 messages 和 steps
- 所有实例完成后，统一回调 onStateChange

### 修改文件

`demo/src/App.tsx`
- 顶部增加 "🔬 对比模式" tab
- 对比模式下的主区域：3 列并排，每列一个 AgentFlow
- 底部统计对比区域

`demo/src/components/AgentFlow.tsx`
- 不需要改动（已有的 AgentFlow 直接复用）

## 技术细节

### 并行执行

- 点击"运行"后，3 个 Agent 实例同时启动
- 各自独立流式渲染（每列独立 AgentFlow）
- 不需要等全部完成才显示，每列各自实时更新
- 可以单独停止某个 Agent

### 底部统计

每列下方显示：
- Tool Call 次数
- 耗时（从开始到结束）
- 使用的工具列表

### 错误处理

- 某个 Agent 失败不影响其他 Agent
- 失败的 Agent 列显示错误信息
- 重试按钮只重试该列

## 不做的事

- ❌ 不要求上传或保存对比结果
- ❌ 不支持自定义对比的 System Prompt（固定 3 种）
- ❌ 不支持 3 个以上模型对比
