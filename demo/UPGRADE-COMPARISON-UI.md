## 改动范围

只改 `demo/src/App.tsx` 中对比模式的渲染部分。

### 总结模式（默认）

在对比任务完成后，自动显示总结模式。包含：

1. **表格状对比卡片**（3 列并排）
   - 每列：策略名 + 颜色标识
   - Tool Call 图标序列（emoji 横排）
   - 调用次数（大字，居中）
   - 耗时
   - 最终回复摘要（前 100 字）

2. **底部对比总结文字**
   - 自动生成："默认策略调用了 X 次工具，激进策略调用了 Y 次，保守策略 Z 次..."
   - 用 Tool Call 次数最多的策略标记为"最多"，最少的标记为"最少"

### 详细模式

点击"详细"tab 后，显示当前的 3 列完整 AgentFlow（和之前一样）。

### 运行中状态

- 运行中默认显示"详细"模式（方便看实时流式更新）
- 运行完成后自动切换到"总结"模式
- 用户可以随时手动切换

## 实现方式

在 App.tsx 中新增一个 `comparisonSubview: 'summary' | 'detail'` 状态。
根据这个状态切换渲染总结卡片还是完整 AgentFlow。
